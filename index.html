Técnologías Frontend

Todo desarrollador Frontend debe conocer HTML, CSS y JavaScript

HTML

Definición

HTML, HyperText Markup Language, es un lenguaje de etiquetas que define la estructura y significado del contenido web.

Sintaxis

<h1> Esto es el texto dentro de una etiqueta h1 </h1>

Conceptos relacionados

  • SEO
  • API del navegador
  • Accesibilidad y Usabilidad

CSS

Definición

CSS, Cascading Style Sheets, es una hoja de estilos donde se le asignan características gráficas a las etiquetas HTML como color, tamaño, posicionamiento, etc

Sintaxis

h1 { color: #0000ff; }/* De esta forma se le asigna el color azul a la letra*/

Conceptos relacionados

  • Responsive Design
  • Preprocesadores (Sass, Less, Stylus, PostCSS, etc)
  • Animaciones
  • Frameworks y librerías (Bootstrap, Foundation, Tailwind, Materialize, etc)

JavaScript

Definición

JavaScript es un lenguaje de programación que permite añadir interactividad al contenido web mediante la lógica del lado del cliente y la manipulación del Document Object Model.

Sintaxis


const x = 123
const y = 321
const z = x + y
if(z < 500) {
console.log(z + ' es menor a 500')
} else {
console.log(z + ' es mayor o igual a 500')
}

Conceptos relacionados

  • DOM
  • APIs
  • Node Package Manager
  • Callbacks, Promises & Aysnc await
  • Aysnc await
  • Paradigmas de progmración (Orientada a objetos, Funcional, Reactiva)
  • Module Bundlers (Webpack, Rollup, Parcel)
  • Patrones de diseño
  • Frameworks y librerias (React, Angular, Vue, Svelte, etc)
